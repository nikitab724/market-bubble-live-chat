# sources AGENTS.md

This directory contains raw or source-like notes used to maintain the LLM Wiki.

Rules:
- Treat source files as mostly immutable evidence.
- If a source needs cleanup, prefer adding a derived wiki page instead of rewriting the source.
- Do not store secrets, private tokens, credentials, or hidden customer data.
- When a source is ingested into durable docs, update `docs/wiki/index.md` and append `docs/wiki/log.md`.
