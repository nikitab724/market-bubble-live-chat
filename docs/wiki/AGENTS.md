# wiki AGENTS.md

This directory owns the persistent navigation and history layer for the LLM Wiki.

Rules:
- `index.md` is the content catalog. Update it when durable docs are added, removed, renamed, or materially repurposed.
- `log.md` is append-only. Add one short entry for repo-changing runs and wiki maintenance passes.
- Keep entries factual and compact.
- Do not store secrets.
