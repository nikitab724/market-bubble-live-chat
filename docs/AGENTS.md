# docs AGENTS.md

This directory is the repo's LLM-maintained wiki.

Rules:
- Read `docs/README.md` before non-trivial source changes.
- Read `docs/wiki/index.md` before changing source or docs.
- Append `docs/wiki/log.md` for repo-changing runs.
- Keep docs concise, current, and linked from `docs/README.md` and `docs/wiki/index.md` when important.
- Update docs in the same run as behavior, setup, deployment, connector, or test changes.
- Keep raw/source material under `docs/sources/` and do not mutate it casually.
- Keep historical planning notes under `docs/superpowers/`; do not rewrite them unless asked.
- Do not include secrets.
- If a new top-level source directory is added, add a matching directory `AGENTS.md`.
